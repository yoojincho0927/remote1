보유기술

  • Spring Framework
    스프링과 스프링부트를 듣고 차이점을 배웠으며 필요 라이브러리를 사용할 수 있습니다.
    Boot사용과 MVC사용이 가능합니다.
  • Java
    자바 Controller와 패키지에 대한 이해와 클래스 사용,
    상속을 이용한 오버라이딩 구현,인터페이스를 통한 다형성 구현이 가능합니다.
  • Bootstrap
    bootstrap에서 제공하는 기본적인 레이아웃과 메뉴, 버튼, 리스트, 탭 등의 다양한 컴포넌트를 사용하여

  • jQuery
    ajax,axios의 이해, css스타일 변형가능, 효과 및 애니메이션을 사용한 경험이 있습니다.
    html axios에서 Controller - Service -mapper등을 거쳐 SQL에서 주고 받는 코드를 작성 한 적이있는데
    post방식과 get방식을 상황에 맞게 사용하는 것이 중요하다고 느꼈습니다. 아직은 좀 더 익숙해져야 할 기술입니다.
  • HTML5
    CSS 선택자를 통해 특정 요소를 선택, 스타일을 적용하거나 변경할 수 있습니다.
    Display의 grid와 flex와 block을 활용하여 UI 구조를 만들고,
    position을 통해 브라우저마다 세밀한 디자인을 경험하였습니다.
  • JavaScript
    이벤트 등록과 호출,정규식을 통한 유효성 검증,css애니메이션 등 가능합니다.
    좀 더 공부를 하여서 TypeScript등 숙련하고 싶습니다.
  • SQL
    기본SQL 문법 사용(DDL, DML, DCL, TCL),Transaction 처리,
    View, Squence, Index의 생성과 사용, 무결성 제약 조건의 이해와 적용,DBeaver를 통한 오라클 클라우드 접속가능합니다.
  • Git
    포트폴리오를 직접 올려본 경험이 있습니다.
    폴더를 만들어 test용으로 수정한 적은 있으나 아직은 좀 더 익숙해져야할 부분입니다.

    어려웠던 점,개선한 점

    • 회원 테이블은 쉽게 만들었으나 가입시 이메일인증 구현을 하고 싶었습니다.
    • MailHandeler,Service 등을 넣어서 작업했고 이를 로그인 서비스에서 boolean check등으로 구현했습니다.
    • 회원가입을 하자마자 바로 이메일 인증을해야 로그인 할 수 있도록 만드는 것이 좀 까다로웠습니다.
      인증키를 저장하기위해 테이블을 수정하기도 했습니다.
    • 로그인 기능을 구현하기 위해 axios post기능을 처음 써보았는데 쉽지 않았습니다.
      아마 그동안 get방식만 써서 그런듯 합니다.
    • 이메일로 아이디 찾기는 구현했으나, 비밀번호 찾기는 어려웠습니다.
      이메일로 임시비밀번호를 생성하고 그 비밀번호 값을 DB에 저장한 후 그 값을 현재 비밀번호에 일치시켜야 수정할수 있도록 만들었는데 그 로직이 쉽지 않았습니다.
    • 좋아요, 스크랩 테이블을 만든 후, 기존 view에서 아이디와 게시판 번호를 insert하는 과정에서 기존 회원테이블의 likeCount,scrapCount부분에서도 업데이트가 되도록 만들어야했는데,이것또한 axios로 받아온 후 @Autowired Board등을 가져와서 해야만 했습니다.

    나의 주요역할

    • WAS 환경 구축 및 도메인 연결
    • 리뷰 기능 DB 서버 구축
    • 리뷰 기능 공통 인터페이스 정의
    • 리뷰 기능 파일 서버 저장,수정,삭제 등 구현
    • 리뷰 사진만보기, 전체보기 구현
    • 리뷰 더보기 등 구현

    어려웠던 점,개선한 점

    • 팀원이 만든 로그인 아이디 세션 값을 리뷰 Controller 에서 받아와서 받아온 값에서 연결 시키는 것이 어려웠습니다.
    • 제가 기존에 만든 리뷰 수정, 삭제 기능에서 아이디 값을 받아오자 경로가 모두 바뀌었는데 그 경로를 찾고 수정하는 점이 까다로웠습니다.
    • 글 수정, 삭제 까지는 가능했지만 문제는 파일서버에 이미지 저장을 하는 것이었습니다.
    • 인터넷에선 파일서버에 저장하는 방법이 많이 나와있었는데 막상 적용시키는 것이 어려웠어서 수업시간에 배운 업로드 코드를 올린 후 수정할 때는 기존 사진을 삭제한 후 다시 insert2하는 방법으로 수정하도록 했습니다.
    • 사진만보기,더보기 등 기능은 sql에서 하지 않고 자바스크립트를 이용했습니다.axios를 받아와서 화면에 출력하는 것이었는데 사진만 보기에서 함수가 많아지자 약간 헷갈렸습니다.
    • 자바스크립트에서도 session값을 불러올 수 있다는 것이 좀 신기했습니다.
    • 개인적으로 axios사용이 조금 어려웠습니다. 이를 보완하고자 사이드 프로젝트를 하나 더 진행했습니다.
